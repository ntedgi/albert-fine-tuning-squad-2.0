## How I Run This Project Locally ?

- clone this repository

- Just Want to Play ?
    - Run : `bash install.sh`
    - Run : `bash run.sh`

 - request: 
    - End-Point : GET `http://localhost:8000/predict`
    - Params : { question : String , paragraphs : Array<String>}
 
 - result:
     - Type : Json
     - Result : {result : [paragraphIndex:Int , Answer:String]
